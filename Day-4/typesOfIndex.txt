1. Single Index:
Its an index created on one field only like name or age or etc. 
MongoDB sorts that field in the index so it can find values faster.

If collection is:
db.users.insertMany([
  { name: "Hiba", age: 22 },
  { name: "Aisha", age: 25 },
  { name: "Sara", age: 20 }
])
Creating a single-field index on age:
db.users.createIndex({ age: 1 })


2. Compound Index:
A compound index is an index on more than one field at the same time, like name + age or category + price.
